# TP_HW2

# Система для управления рецептами

Проект представляет собой систему для управления рецептами со следующими возможностями: создание рецептов, масштабирование порций, создание списка покупок, создание диетических рецептов, управление ингридиентами. 

# Использование 

Запуск кода:
Для запуска демонстрационного кода введите в терминале из корневой папки команду python main.py

Запуск тестов: 
Для запуска тестов, находясь в корневой папке введите в терминале команду pytest test_recipes.py -v
